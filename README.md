# 格式转换工具 - 系统说明

## 一、软件简介

格式转换工具是一款用于**档案数字化**的桌面软件，可以将扫描的图片（JPG、PNG等）转换为标准的PDF文件，同时支持Word、Excel等Office文档的转换。

### 1.1 主要功能

| 功能 | 说明 |
|-----|------|
| **图片转PDF** | 将扫描的图片转换为PDF文档 |
| **双层PDF** | 转换后的PDF可以搜索、复制文字 |
| **单层PDF** | 纯图片格式，无法搜索文字 |
| **批量转换** | 一次性处理整个文件夹的文件 |
| **自动整理** | 按档号规则自动创建文件夹结构 |
| **PDF合并** | 将多个文件合并成一个PDF |
| **档案著录** | 提供档案数据著录与关联功能 |

### 1.2 适用场景

- 📁 **档案馆/档案室**：纸质档案数字化加工
- 🏢 **企事业单位**：合同、票据等文件电子化
- 📚 **图书馆/博物馆**：资料数字化保存
- 👤 **个人用户**：批量整理扫描文件

## 二、核心概念说明

### 2.1 什么是双层PDF？

**双层PDF** = 图片层 + 文字层

| 层级 | 作用 | 特点 |
|-----|------|------|
| **图片层** | 显示原始扫描图像 | 保持原始清晰度 |
| **文字层** | 存放OCR识别的文字 | 可以搜索、复制、选中 |

> 💡 **档案标准要求**：正规档案数字化都需要使用双层PDF，方便后续检索和查阅。

### 2.2 什么是单层PDF？

只有图片层，没有文字层，相当于把内容变成一张纯图片。

**适用情况**：
- 需要防止文字被复制的保密文件
- 不需要搜索功能的纯图像存档

### 2.3 什么是档号？

档号是档案的唯一标识编号，通常格式为：

```
全宗号-档案门类-年度-保管期限-机构号-件号
```

**示例**：`J001-WS-2016-Y-BGS-001`

| 部分 | 含义 | 示例 |
|-----|------|------|
| 全宗号 | 档案所属单位 | J001 |
| 档案门类 | 档案类型 | WS（文书） |
| 年度 | 年份 | 2016 |
| 保管期限 | 保存时间 | Y（永久） |
| 机构号 | 产生部门 | BGS（办公室） |
| 件号 | 顺序编号 | 001 |

## 三、功能详解

### 3.1 文件转换

#### 支持的文件格式

| 类型 | 格式 |
|-----|------|
| **图片** | JPG、JPEG、PNG、BMP、TIFF |
| **PDF** | 已有PDF可进行OCR识别 |
| **Word** | DOC、DOCX |
| **Excel** | XLS、XLSX |
| **PPT** | PPT、PPTX |

#### 两种输入方式

**方式一：选择文件**
- 适合：转换几个零散文件
- 操作：点击「选择文件」按钮，手动选择要转换的文件

**方式二：选择目录**
- 适合：批量转换整个文件夹
- 操作：点击「输入目录」，选择包含文件的文件夹

#### 四种输出模式

| 模式 | 说明 | 适用场景 |
|-----|------|---------|
| **双层PDF** ⭐推荐 | 带可搜索文字层 | 档案数字化标准格式 |
| **单层PDF** | 纯图片，无文字层 | 防复制、纯图像存档 |
| **合并PDF** | 多文件合并成一个 | 按卷/册整理档案 |
| **拆分PDF** | 按范围提取或逐页拆分 | 多页文档拆分及提取 |

### 3.2 文件夹规则（规则0）

软件仅保留输出文件夹规则 **规则0**：

#### 规则0：不创建文件夹（默认）
```
输入：D:\输入\001.jpg
输出：D:\输出\001.pdf
```
⚠️ 注意：所有转换后的文件直接输出到指定的根输出目录中，不创建任何子文件夹。如果有同名文件，会发生相互覆盖的情况。系统在开始转换前会进行冲突检测并给予警告提示。

## 四、使用流程示例

### 4.1 档案数字化标准流程

```
第1步：整理文件
   按档号规则整理好文件夹
   例如：J001/WS/2016/Y/BGS/001/

第2步：选择输入目录
   点击「输入目录」选择 J001 文件夹

第3步：设置参数
   输出模式：双层PDF
   文件夹规则：规则4
   PDF命名来源：最后一层文件夹名

第4步：开始转换
   点击「开始转换」等待完成

第5步：检查结果
   在输出目录中查看生成的 PDF 并核对数量
```

### 4.2 快速转换单个文件

```
第1步：选择「文件上传模式」
第2步：点击「选择文件」选择图片
第3步：选择「双层PDF」
第4步：选择「规则0」
第5步：开始转换
```

## 六、常见问题

### Q1：双层PDF和单层PDF有什么区别？

| 对比项 | 双层PDF | 单层PDF |
|-------|--------|--------|
| 能否搜索文字 | ✅ 能 | ❌ 不能 |
| 能否复制文字 | ✅ 能 | ❌ 不能 |
| 文件大小 | 较小 | 较大 |
| 档案标准 | 符合 | 不符合 |

### Q2：转换后的文件名不对？

- **选择文件模式**：使用原文件名
- **选择目录模式**：根据「PDF命名来源」设置决定
- 建议使用「最后一层文件夹名」作为文件名

### Q3：为什么有些规则选不了？

规则3、4、6、7只能在「选择目录」模式下使用，请先选择输入目录。

### Q4：合并PDF功能用不了？

合并PDF只能在「选择目录」模式下使用，且文件夹中应是同类型文件。

### Q5：转换过程中可以关闭软件吗？

❌ 不可以。转换过程中请保持软件运行，否则可能导致转换失败。

### Q6：Word转PDF后还能编辑吗？

- 双层PDF：可以复制文字，但不能直接编辑
- 如需编辑Word，请保留原始文件

## 七、注意事项

### 7.1 使用前

1. ⚠️ **建议先测试**：正式转换前，先用几个文件测试输出效果
2. ⚠️ **备份原文件**：转换前建议备份原始文件
3. ⚠️ **检查文件名**：规则0模式下，同名文件会互相覆盖

### 7.2 使用中

1. ⏳ **耐心等待**：大文件或批量转换需要较长时间
2. 💾 **保持运行**：转换过程中不要关闭软件
3. 🔋 **避免断电**：确保电脑不会意外关机

### 7.3 使用后

1. ✅ **检查输出**：打开几个PDF检查转换效果
2. 📊 **核对统计**：使用统计功能核对文件数量
3. 📝 **保存记录**：如需留存，导出统计报表
